# Auth Project

This project contains simple HTML pages for authentication.

## How to View

1. Open `login.html` directly in any web browser to view the login page.
2. You can also view `gayatri.html` or `success.html` similarly.
3. No server is required to view these static files.

## Code Explanation

- **`login.html`**: A static HTML file with a login form styled with Bootstrap. It contains inline JavaScript to validate credentials (`admin`/`admin`) and redirects to `success.html` on success using `window.location.href`.
- **`success.html`**: A simple page displayed after successful login.
