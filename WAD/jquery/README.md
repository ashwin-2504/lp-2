# jQuery Project

This is a simple project using jQuery.

## How to View

1. Open `index.html` directly in any web browser to view the project.
2. No server is required to view these static files.

## Code Explanation

- **`index.html`**: Uses jQuery Mobile to create a mobile-friendly login page.
- It handles the click event on the submit button, reads the values, and checks for `admin`/`admin`.
- It displays a message and an alert on success.
